TourDeForce is a portfolio to-do list app. It uses Gamification through awards, to make completion of projects and their items more compelling.


(Temporary design)

Authors and acknowledgment

The app is based on the Ultimate Portfolio App course from Paul Hudson (@twostraws) at Hacking with Swift.
I would like to thank Paul for not only guiding us through his part of the code, but also for the amazing contributions (in both scope an clarity) that he has made to the Swift community.

All other code is written by the maintainer of this repository, Tor Rafsol Løseth (@loseth).

Knowledge applied

Technologies

  • iOS (iPadOS, macOS and watchOS will follow)
  • Foundation, Swift and SwiftUI.

Concepts

  • Accessibility with VoiceOver.
  • Architecture navigation with TabView and NavigationView/NavigationLink.
  • Archiving objects with Codable and JSONDecoder
  • Filter and sort objects with NSCompundPredicate, NSPredicate, NSSortDescriptor.
  • Internationalization (i18n) and Localization (i10n) with LocalizedStringKey.
  • Persistent storage with Core Data
  • SwiftUI data flow and state with @Environment, @EnvironmentObject, @Fetchrequest, @ObservedObject, @SceneStorage, @State, @StateObject, and ObservableObject.
  • Unit testing with XCTest.

Tools

  • Image editing with Acorn
  • Integrated development environment with Xcode

  • Project documentation with Jazzy
  • Screenshots with ControlRoom
  • Swift style with SwiftLint
  • Text processing with BBEdit
  • Version control with Git
